# ADUMI Frontend

Interface web para o sistema de vendas ADUMI, consumindo a API REST do backend.  
Permite gerenciar produtos, clientes, vendas e itens de venda de forma interativa e amigável.

---

## Tecnologias Utilizadas

- Vue.js 3  
- Vite  
- Axios  
- CSS  

---

## Funcionalidades

- Listar, criar, atualizar e excluir **produtos**.  
- Listar, criar, atualizar e excluir **clientes**.  
- Registrar **vendas** e visualizar seus **itens**.  
- Mostrar detalhes de produtos e clientes associados a cada venda.  

---

## Configuração e Teste

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/adumi-frontend.git
   cd adumi-frontend

2,Instalar dependências:
npm install

3.Rodar servidor de desenvolvimento
npm run dev

4.Acesse o frontend no navegador:
http://localhost:5173


5.Certifique-se de que o backend (adumi-backend) está rodando na porta 8000 para o frontend consumir a API.
